Spy Print Configurations

Print Options

please note that you can connect only using websocket

Please specify the host and event listener name

Example: https://your-domain.com or wss://your-domain.com

please note that object data sent to the socket should look like the following:

{fileUrl: https://example.com/your/file/path, machineNo: machine1}

This option allows you to send the print job to specific machine1 if the file is generated on the server
